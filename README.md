# Gobby's Workshop

A prototype 2D platformer built with Godot Engine.

## Purpose

This project was created as a learning and portfolio project to demonstrate core platformer mechanics and game development skills in Godot.

Rather than being a complete game, it serves as a prototype showcasing player controls, enemy behaviour, UI systems and level progression.

## Features Implemented

### Player

* Character movement
* Jumping
* Health system
* Death and respawn flow

### Enemies

* Enemy detection
* Chase behaviour
* Damage dealing
* Knockback effects
* Defeating enemies by jumping on them

### Gameplay Systems

* Collectible gears
* Progress tracking
* Level completion conditions
* Intro dialogue
* Game Over screen
* Victory screen

### User Interface

* Health bar
* Gear counter
* Progress display

## Technologies

* Godot Engine
* GDScript
* Pixel Art Assets

## Learning Goals

This prototype was developed to practice:

* Scene architecture in Godot
* GDScript programming
* State-based gameplay logic
* Collision handling
* UI implementation
* Enemy AI behaviour
* Object interaction systems
